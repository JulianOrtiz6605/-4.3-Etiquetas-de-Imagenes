* La etiqueta img no tiene etiqueta de cierre.
* El atributo src="" hace referencia a la ruta donde esta la imagen que vamos a utilizar.
* El atributo alt="" Se usa para mostrar un texto relacionado a la imagen en caso que esta no se pueda mostrar.
* El atributo title="" Se usa para mostrar un texto cuando el usuario coloque el cursor encima de la imagen.
* El atrubuto width="" y height="" Se usan para darle un ancho y una altura a la imagen.(Es recomedable usar una de estas dos para no distorsionar la imagen.
* Para colocar el valor de height o width en pixeles puede ser solo el número o su valor en pixeles asi:width="500" o width="500px" también hay otros formas de medida.
